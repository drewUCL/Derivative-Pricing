Derivative Pricing <br>
=============================================

###Introduction
This code allows the user to price **European Put Options** on the follwoing contracts: 

1. UNDERLYING ASSET WITH NO DIVIDENDS
2. ASSET WHICH PAYS A DIVIDEND
3. ON FOREIGN CURRENCY BOND WITH INTEREST RATE (RF)
4. ON COMMODITY WITH STORAGE COST
5. ON A FUTURES CONTRACT
